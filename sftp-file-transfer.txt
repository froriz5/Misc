Use sftp to transfer files to remote server
1) open up sftp connection to remote server: sftp <remoteServer>
2a) check local working dir: 'lpwd'
2b) check remote working dir: 'pwd'
3a) list files on remote: 'ls'
3b) list files on local: 'lls'
4) upload a file: 'put local.*'
5) upload multiple files: 'put *.txt'
6) exit sftp shell: '!'
